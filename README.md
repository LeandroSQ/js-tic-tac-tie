## How to play

Just click on the cells on the 3x3 grid, each click consists in one turn.

The first player to get three connected cells in a straight line wins.

## Project features

- Light and Dark mode
- Mobile support
- Vanilla JS
- Player vs Player
- Player vs AI
- Sound effects
- Animations
